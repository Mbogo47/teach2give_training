Everyday Git

What is Git

Git is a version control system (VCS) that allows developers to track changes in their codebase, collaborate with others, and maintain a history of their project.

A version control system is a tool that records changes to files over time so that specific versions can be recalled later. This is especially helpful when working in teams, as it provides the ability to manage contributions without overwriting each other's work.

Other popular version control systems include:

  • Subversion (SVN)
  • Mercurial
  • Bazaar

However, this blog will focus on Git, and we’ll also touch on GitHub, a popular hosting platform for Git repositories.

Github

GitHub is a hosting platform for git repositories.

There are tons of competing tools that provide similar hosting and collaboration features including GitLab, BitBucket etc.

Why Use GitHub

  • Collaboration - GitHub makes it easier to work on a project with other people.
  • Open source projects - GitHub is the home of open source projects on the internet. Projects such as React, Postgresql, tensorflow et.c are hosted on GitHub, if you plan to contribute to open source, knowledge of GitHub is essential.
  • Exposure - your GitHub profile showcases your own projects and contributions to other projects. It can act as a resume that employers will consult when they want to hire you.
  • Stay up to date- being active on GitHub is the best way to stay up to date with trends in your favorite language or framework or tool.
